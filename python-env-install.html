[安装] Python相关

2013 七月 06 周六 Zoey Young Python Windows Python pip virtualenv

重装系统后, 重新配置了下Python环境, 并不复杂, 但是得打开好几个网站, 我习惯先看下官网有啥更新说明, 因此记录下, 以免每次都要找

安装Python, 下载安装Python 2.7.X, 安装目录C:\Python27

安装setuptools, Windows下, 下载ez_setup.py文件并运行

在环境变量PATH中追加: C:\Python27;C:\Python27\Scripts 或 C:\Python33;C:\Python33\Scripts

安装pip, 可以在Github上下载开发版, 然后解压, 进入根目录, 使用命令行安装

python setup.py insatll

使用requirements.txt

pip install -r requirements.txt

安装virtualenv

pip install virtualenv

或最新开发版:

pip install https://github.com/pypa/virtualenv/tarball/develop

安装Sphinx, 用于生成文档

pip install sphinx

http://stackoverflow.com/questions/4750806/how-to-install-pip-on-windows

Unofficial Windows Binaries for Python Extension Packages

lxml
